Esta pagina web fue creada para diseñar y desarrollar los elementos para la construcción de un sitio web utilizando html y css.

PROPOSITO

El proposito de este sitio es analizar para que funciona las etiques basicas de html y los atributos y valoes que componen un sitio web, no obstante debemos comprender para que sirve cada etique basicas y como llevarlo a la practica.
